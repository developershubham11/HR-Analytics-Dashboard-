# HR Analytics Dashboard

## 📌 Project Overview
This project is an interactive HR Analytics Dashboard built using Power BI, SQL, and Excel.

## 🛠️ Tools Used
- Power BI
- SQL
- Excel

## 📊 Dashboard Features
- Employee Overview
- Attrition Analysis
- Department-wise Analysis
- Salary Slab Analysis
- Gender Analysis
- Age Group Distribution
- Experience Analysis
